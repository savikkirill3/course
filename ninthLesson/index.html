Девятый урок
  1. Реализовать функцию getElementsById(id), который будет возвращать массив тегов всех элементов на странице, у которых id равен id. Если теги в массиве повторяются, удалить их повторные.

  2. Написать функцию whichMouseButtonPressed(), которая будет вызываться каждый раз, когда человек будет нажимать что либо мышью на странице и возвращать информацию, какой кнопкой мыши было нажатие.

    • Левая кнопка мыши: “left mouse button”
    • Правая кнопка мыши: “right mouse button”
    • Средняя кнопка (колесико): “middle mouse button”

  3. Переписать функцию getElementsById(id) из первого задания, которая будет работать асинхронно и возвращать promise.

    Пример:

    • getElementsById(“container”)
    • .then((res) => console.log(`Result: ${res}`))
    • .catch((err) => console.log(`Error: ${err}`));
